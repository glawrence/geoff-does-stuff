This file demonstrates different types of list available in HTML. Sometimes a table actually works better but for when it does not here are some lists.

With the Ordered and Unordered lists you can use text formatting techniques to highlight the title or key part, for example <strong>Item One</strong>. It is worth commenting here that the "strong" HTML element is for emphasis and whilst commonly rendered as bold, does not mean bold, if you really want bold then use proper text formatting, which for bold is font-weight in CSS (Cascading Style Sheets).

The Unordered List

  • Item One: This is the full description of the first item.
  • A Second Item: As the second item this needs a nice descrption.
  • Third item: However this time we need a little description.

The Ordered List

  1. Item One: This is the full description of the first item.
  2. A Second Item: As the second item this needs a nice descrption.
  3. Third item: However this time we need a little description.
